# GitHub Explorer

GitHub Explorer is a browser-based tool for searching and visualizing any GitHub profile. Enter a username to instantly view their public stats, repositories, languages, contributions, pull requests, issues, followers, and more - all in one clean interface.

No sign-up. No ads. Completely free to use.

# Features

Profile Overview - View location, website, Twitter, company, account creation date, followers, following, and public repo count

Repositories - Browse all public repositories with filtering by stars, name (A-Z / Z-A), and forks

Pinned Repositories - See the repositories the user has chosen to highlight

Languages - Visual breakdown of programming languages used across repositories

Contributions - Contribution activity overview

Pull Requests - Detailed pull request stats split by Merged, Open, Closed, and Total

Issues - Issue stats split by Open, Closed, and Total

Starred Repos - List of repositories the user has starred

GitHub Token Support - Optionally add a personal GitHub token to increase API rate limits

# How to Use

1. Open the app in your browser
2. Type any GitHub username into the search bar
3. Press Enter or click Search
4. Browse the profile - use the sidebar sections to explore repositories, languages, pull requests, issues, and more
5. Optionally add a GitHub token via the token button to avoid hitting API rate limits
